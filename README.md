# MPU6050 姿态解算原理

## 一、解算方案概述

本工程采用 **Mahony 互补滤波算法**，以四元数为姿态表示形式，融合三轴加速度计与三轴陀螺仪数据，输出 Roll（横滚）、Pitch（俯仰）、Yaw（偏航）三个欧拉角。

MPU6050 为六轴惯性测量单元，不含磁力计，因此：
- **Roll 与 Pitch**：由加速度计观测重力方向进行长期修正，无累积漂移
- **Yaw（航向角）**：完全依赖陀螺仪 Z 轴积分，加速度计无法修正，存在长期温漂

---

## 二、传感器数据预处理

### 2.1 零偏校准
上电初始化阶段，在传感器静止状态下采样数百组数据，取平均值作为陀螺仪与加速度计的零偏偏移量。后续每次读取原始数据时减去该零偏，消除出厂安装误差与温漂带来的静态偏移。

### 2.2 量程换算
- 加速度计：原始 16 位整型除以灵敏度系数（±2g 量程对应 16384 LSB/g），得到单位为 g 的加速度值
- 陀螺仪：原始 16 位整型除以灵敏度系数（±250dps 量程对应 131 LSB/(°/s)），得到单位为 °/s 的角速度值，再乘以 π/180 转换为 rad/s

### 2.3 死区处理
对陀螺仪三轴角速度设置死区阈值（0.25 dps），小于阈值的角速度直接置零，抑制静止状态下的量化噪声与微小零飘，避免静止时角度缓慢累积。

---

## 三、Mahony 互补滤波原理

### 3.1 互补滤波核心思想
陀螺仪短期精度高、动态响应快，但积分后会累积漂移；加速度计长期稳定（重力方向恒定），但动态抗干扰差。互补滤波将两者加权融合：
- 高频段（动态变化）信任陀螺仪
- 低频段（稳态）信任加速度计

### 3.2 误差计算
根据当前四元数推算机体坐标系下的理论重力方向矢量，与加速度计实测的重力方向矢量做**叉乘运算**，叉乘结果的大小与方向直接反映姿态误差：
- 误差矢量的方向代表需要修正的旋转轴
- 误差矢量的模长代表倾斜角度大小

### 3.3 PI 控制器修正
将误差矢量输入 PI 控制器，输出修正角速度叠加到陀螺仪原始数据上：
- **比例项（Kp = 2.0）**：快速拉回姿态偏差，决定加速度计的信任权重
- **积分项（Ki = 0.005）**：消除陀螺仪零偏导致的稳态误差，积分结果限幅防止饱和

修正后的角速度同时具备陀螺仪的动态响应速度与加速度计的长期稳定性。

## 四、四元数更新

### 4.1 四元数微分方程
采用一阶龙格-库塔法对四元数进行数值积分，核心公式为：

$$\dot{q} = \frac{1}{2} q \otimes \omega$$

其中 ω 为修正后的三轴角速度矢量，⊗ 为四元数乘法。每帧乘以时间间隔 dt 后累加到当前四元数上，完成姿态更新。

### 4.2 四元数归一化
每次积分后对四元数进行归一化处理，保证四元数模长恒为 1。由于数值积分存在截断误差，长期运行会导致模长漂移，归一化是保证姿态解算稳定性的必要步骤。

本工程使用快速逆平方根算法（卡马克算法）加速归一化计算，减少 MCU 运算开销。

---

## 五、四元数转欧拉角

### 5.1 转换公式
更新后的四元数通过以下公式转换为 ZYX 内旋顺序的欧拉角：

- **Roll（横滚角）**：绕 X 轴旋转角度，由四元数 q0、q1、q2、q3 组合计算
- **Pitch（俯仰角）**：绕 Y 轴旋转角度，含 ±1 限幅保护，避免万向节锁时 asin 函数越界
- **Yaw（偏航角）**：绕 Z 轴旋转角度，即航向角

### 5.2 Yaw 角计算特点
Yaw 角由四元数中的 q0、q1、q2、q3 共同通过反正切函数求得。由于六轴传感器缺少磁力计对绝对航向的观测，Mahony 滤波的误差修正项对 Yaw 轴不起作用——加速度计无法感知绕重力轴的旋转，因此 Yaw 角本质上仍是纯陀螺仪积分的结果。

---

## 六、Yaw 角漂移分析

### 6.1 漂移来源
1. **陀螺仪零偏温漂**：环境温度变化导致陀螺仪零偏缓慢偏移，经积分后持续累积为角度误差
2. **量化噪声累积**：ADC 量化噪声与随机游走噪声，长时间积分后放大
3. **G 敏感误差**：加速度变化对陀螺仪输出产生的交叉耦合干扰

### 6.2 漂移量级
常温静止状态下，典型漂移速率约为 0.5°/分钟，短时间（2~3 分钟内）的小车赛程中漂移量可忽略，不影响巡线控制精度。

### 6.3 工程应对
- 每次发车前执行零点复位，消除历史累积误差
- 巡线过程使用快速版解算（仅陀螺仪积分），减少 I2C 总线占用
- 进入菜单或静止状态时切换完整版解算，修正 Roll/Pitch 累积误差

---

## 七、零点复位机制

通过软件偏移量实现 Yaw 角清零，复位操作将四元数重置为单位四元数（1, 0, 0, 0），对应欧拉角全为 0°，同时清零积分项与时间戳基准。复位后以当前车体朝向为新的航向零点。
