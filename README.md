# MPU6050 Yaw 姿态解算 — MSPM0G3507 实现

> 基于 Mahony 互补滤波 + 独立 Yaw 梯形积分，为 TI 电赛巡线小车提供航向辅助。

---

## 一、方案概述

| 项目 | 说明 |
|------|------|
| **算法** | Mahony 互补滤波（四元数）+ 独立 Yaw 梯形积分 |
| **传感器** | MPU6050 六轴 IMU（I2C1, 400kHz），无磁力计 |
| **MCU** | MSPM0G3507, 80MHz |
| **Roll/Pitch** | 四元数 → 加速度计 PI 修正 → 长期无漂移 |
| **Yaw** | 陀螺仪 Z 轴独立梯形积分 → 无绝对参考，短期可用 |
| **两种模式** | 完整版（~0.8ms）用于静止校准 / 轻量版（~0.15ms）用于巡线 |

---

## 二、传感器预处理

### 2.1 零偏校准

`MPU6050_Calibrate(200)` — 上电后传感器保持静止，采样 200 组数据取均值：

- **陀螺仪零偏**：三轴原始值的算术平均 = 静止输出偏置
- **加速度计零偏**：X/Y 取均值，Z 轴以 1g 为参考（均值 − 灵敏度）

后续每次读取：`校准值 = 原始值 − 零偏`

### 2.2 量程选择

| 传感器 | 量程 | 灵敏度 | 原因 |
|--------|------|--------|------|
| 陀螺仪 | **±2000 dps** | 16.4 LSB/°/s | 直角转弯角速度可达 300~400°/s，±250 会饱和切峰 |
| 加速度计 | ±2g | 16384 LSB/g | 低速小车加速度不大，±2g 分辨率足够 |

### 2.3 死区滤波

三轴角速度绝对值 < 0.25°/s 时直接置零，抑制静止状态的量化噪声。

```c
if (gz > -0.25f && gz < 0.25f) gz = 0.0f;
```

## 三、Mahony 互补滤波（Roll/Pitch）

### 3.1 核心思想

```
陀螺仪：短期精准、动态响应快 → 积分漂移
加速度计：长期稳定（重力恒定） → 动态干扰大

互补：高频信陀螺，低频信加速度
```

### 3.2 误差计算

1. 当前四元数推算理论重力方向：`v_est = q ⊗ g ⊗ q*`
2. 加速度计实测重力方向：`a_meas = a / |a|`
3. 叉乘 = 姿态误差：`error = a_meas × v_est`

叉乘方向即需要修正的旋转轴，模长反映倾斜程度。

### 3.3 PI 修正

```
修正角速度 = Kp × error + Ki × ∫error·dt
Kp = 2.0   —— 快速拉回偏差
Ki = 0.005 —— 消除稳态零偏（积分限幅 ±0.3 防饱和）
```

### 3.4 为什么 Yaw 不受修正

加速度计只能感知重力方向——车绕重力轴水平旋转时，重力矢量在机体坐标系的投影不变，叉乘得到的 `ez` 分量恒为零。因此 Mahony 修正对 Yaw **不起作用**，Yaw 必须独立积分。

## 四、四元数积分

### 4.1 微分方程（一阶龙格-库塔 / 欧拉法）

$$\dot{q} = \frac{1}{2} q \otimes \omega$$

展开为标量形式：

```
q0 += -½·dt·(q1·gx + q2·gy + q3·gz)
q1 +=  ½·dt·(q0·gx + q2·gz − q3·gy)
q2 +=  ½·dt·(q0·gy − q1·gz + q3·gx)
q3 +=  ½·dt·(q0·gz + q1·gy − q2·gx)
```

### 4.2 归一化

每次积分后用快速逆平方根（Carmack 算法，两次牛顿迭代）归一化，防止模长漂移。

### 4.3 转欧拉角（ZYX 内旋）

```
roll  = atan2(2q2q3 + 2q0q1, 2q0² + 2q3² − 1) × 180/π
pitch = −asin(2q0q2 − 2q1q3) × 180/π       （含 ±1 限幅防越界）
yaw   = 独立梯形积分（不通过四元数转换）
```

## 五、Yaw 独立梯形积分

### 5.1 为什么独立

加速度计不能观测绕重力轴的旋转 → Mahony 修正对 Yaw 无效 → Yaw 用独立梯形积分，不经过 Mahony PI 环节。

### 5.2 梯形积分 vs 欧拉积分

```
欧拉: yaw += ω(t) × dt           矩形，O(dt²) 误差
梯形: yaw += ½[ω(t−1)+ω(t)]×dt   梯形，O(dt³) 误差
```

### 5.3 保护机制

| 保护 | 值 | 作用 |
|------|-----|------|
| 死区滤波 | < 0.25°/s 清零 | 静止噪声不积分 |
| dt 限幅 | ≤ 50ms | 卡顿防跳变 |
| 角速度限幅 | ±500°/s | 剔除 I2C 通信野值 |
| 单帧增量限幅 | ±5° | 防异常跳变 |
| 归一化 | ±180° 折返 | 数值不溢出 |
| 温漂自补偿 | α = 0.001 | 静止时微调零偏 |

### 5.4 温漂自补偿

检测到车静止（`|a| ≈ 1g` 且三轴角速度 < 0.5°/s）时，以千分之一的步长微调 Z 轴零偏。时间常数 ~1000 秒，不会把真实缓慢转动误认为温漂。

## 六、两种运行模式

| | 完整版 `QuaternionUpdate` | 轻量版 `QuaternionUpdateFast` |
|---|---|---|
| **I2C 读取** | 加速度 6B + 陀螺 6B + 温度 2B | 仅陀螺 6B |
| **加速度修正** | ✅ PI 修正 Roll/Pitch | ❌ 无 |
| **Yaw** | 梯形积分（两种一样） | 梯形积分 |
| **耗时** | ~0.8ms | ~0.15ms |
| **用途** | 静止校准、菜单页 | **巡线实时** |

---

## 七、零点复位

`MPU6050_QuaternionReset()` — 将四元数重置为单位四元数 `(1,0,0,0)`，Yaw 归零，清除 PI 积分项。复位后以当前车体朝向为新零度。

---

## 八、漂移与实际精度

| 误差来源 | 量级 | 应对 |
|----------|------|------|
| 陀螺零偏温漂 | ~0.5°/min | 温漂自补偿 + 跑前归零 |
| 量化噪声 | < 0.1°/s | 死区滤波 |
| 直角转弯 | < 2° | ±2000dps 防饱和 + 梯形积分 |

巡线闭环控制周期 < 10ms，秒级漂移可忽略，一场比赛（2~3 分钟）内足够使用。
